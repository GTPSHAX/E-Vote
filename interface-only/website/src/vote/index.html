Peraturan

Umum

  1. Peserta dan Petugas diharapkan saling mejaga kerahasiaan suara.
  2. Peserta dan Petugas dilarang menimbulkan kegaduhan di dalam ruangan.

E-Vote

  1. Peserta dan Petugas dilarang keras untuk memodifikasi atau menggunakan tool apapun pada Website ini.
  2. Peserta diwajibkan melapor ke Petugas jika terjadi kendala.

Daftar Paslon

Cara melakukan vote:
  1. Membaca Visi, Misi, dan Harapan dari setiap Calon.
  2. Setelah itu, Peserta dapat memilih salah satu Calon.
  3. Peserta dapat memberikan pesan kepada Calon yang mereka pilih.
  4. Dan terakhir, Peserta dapat menekan tombol Vote untuk mengirimkan data.
